# NerdQuiz — 6-Minute Video Script (Storytelling Walkthrough)

**Format:** Screen recording with voiceover narration
**Total runtime:** 6:00
**Audience:** Chapter 5 review / public demo
**Style:** Formal, cinematic storytelling — follow a student's journey through the platform

---

## VISUAL STYLE GUIDE

**Narration tone:**
- Warm, measured, confident — like a documentary narrator
- Each section opens with a problem or question, then reveals the solution
- Transitions connect scenes with narrative threads, not abrupt cuts

**Pacing:**
- 15–20 seconds per major feature reveal
- Pauses after key moments for visual absorption
- Music swells at emotional peaks (quiz success, exam completion)

**Music:**
- Ambient piano + soft synth pad throughout
- Rising tension for exam simulation
- Triumphant swell for the outro

## Key Talking Points

1. **Structured learning** — The Learning Map mirrors the ITPEC FE syllabus with five milestone categories and twenty topic stops
2. **Instant feedback** — Every quiz answer comes with an explanation, not just a correct/wrong indicator
3. **Growth from mistakes** — The Mistake Garden turns wrong answers into learning opportunities
4. **Real exam conditions** — The simulation mode provides sixty-question timed sessions with hearts
5. **Data-driven guidance** — Weak point analysis tells students exactly where to focus
6. **AI-powered content** — Three agents handle extraction, validation, and generation
7. **Eight developers, two weeks** — The platform was built in half the time with AI amplification
