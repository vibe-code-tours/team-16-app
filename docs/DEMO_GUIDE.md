# Demo Guide — NerdQuiz

This guide explains how to present the current state of the project for the Chapter 5 review.

## 🎯 Demo Goals
Showcase the **"Core Learning Loop"**: the seamless transition from discovery $ightarrow$ learning $ightarrow$ testing $ightarrow$ reflection.

## 🎬 Step-by-Step Walkthrough

### 1. The Entry (Discovery)
- **Action**: Start at the Landing Page.
- **Narrative**: "Welcome to NerdQuiz, the gamified way to master the ITPEC FE exam."
- **Action**: Click "Get Started" $ightarrow$ Register/Login.

### 2. The Map (Navigation)
- **Action**: Navigate to the Learning Map.
- **Narrative**: "Here is the Learning Map. It organizes the massive FE syllabus into bite-sized topics. We've already seeded core areas like OS and Networking."
- **Action**: Hover over a topic card to show the hover effect.

### 3. The Lesson (Learning)
- **Action**: Click into a topic $ightarrow$ Select "Lesson 1".
- **Narrative**: "Before every quiz, students read a short, 2-minute lesson. This ensures they have the foundational knowledge before being tested."
- **Action**: Scroll through the lesson content.

### 4. The Quiz (Testing)
- **Action**: Return to Topic Detail $ightarrow$ Start Quiz.
- **Narrative**: "Now we test that knowledge. The quiz provides instant feedback. If the user is correct, they earn XP. If they are wrong, the mistake is recorded."
- **Action**: Answer one correctly, then answer one incorrectly.

### 5. The Garden (Reflection)
- **Action**: Click "Mistake Garden" in the header.
- **Narrative**: "This is the heart of our app: the Mistake Garden. Instead of just forgetting a wrong answer, it's stored here as a 'seed'. The student can return here to review the correct answer and 'clear' the mistake."
- **Action**: Mark a mistake as "Reviewed".

### 6. The Profile (Progression)
- **Action**: Return to the Map.
- **Narrative**: "Notice the XP has increased. This gamification keeps students motivated to complete the entire curriculum."

- **Supabase RLS**: Mention that user data is secure and private.
