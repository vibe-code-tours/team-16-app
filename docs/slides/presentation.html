ITPEC FE Certification Prep

NerdQuiz

A gamified learning platform that turns IT exam preparation into an engaging, structured journey — lessons, quizzes, review, and exam simulation in one loop.

Full Platform Walkthrough · 15 pages · Built with React, Spring Boot, Supabase

01 / Public

First Impression:
Landing Page

Dark-themed hero with the NerdQuiz owl mascot. Clear value proposition and two CTAs — "Start learning free" and "Continue your journey."

  • Gamified IT exam prep tagline
  • Today's Mission card with progress stats
  • 6 feature cards showcasing the learning loop
02 / Auth

Sign In
Simple & Secure

Supabase Auth powers the login flow. Email/password authentication with Google OAuth option. JWT tokens verified by the Spring Boot backend.

  • Email + password authentication
  • Google OAuth one-click sign-in
  • JWT verification on every backend request
03 / Auth

Create Account
Start Your Journey

One-step registration. The user profile is auto-created on first login — no separate profile setup needed.

  • Password confirmation validation
  • Auto profile creation on first login
  • Redirects to Learning Map after signup
04 / Core

Learning Map
Your Roadmap

The central hub. Five milestone categories — Technology, Security, Management, Strategy, Business — with 20 topic stops. Clear each to unlock the next.

  • XP counter and day streak in header
  • Continue-where-you-left-off CTA
  • Progress bar with journey stats
  • Locked/unlocked/in-progress states
05 / Quiz

Quiz Listing
Pick Your Challenge

Browse all available topic quizzes. Each card shows the topic name, question count, and your current score. Jump straight into any unlocked quiz.

  • Grid layout with topic cards
  • Score tracking per quiz
  • Quick access to any available quiz
06 / Topic

Topic Detail
Lessons + Quiz

Each topic has bite-sized lessons to study first, then a 5-question quiz to prove understanding. Complete lessons to unlock the quiz.

  • Sequential lesson unlock
  • 5 min per lesson · 10 XP each
  • Quiz button appears after lessons
  • Completion checkmarks for done items
07 / Lesson

Bite-Sized Lessons
Focused Learning

Short, readable lessons designed for 5-minute study sessions. Content is structured with headings, tips, warnings, and examples for clarity.

  • Markdown-rendered content
  • Progress tracking per lesson
  • Mark-as-complete to unlock next
08 / Quiz

Practice Quiz
Earn XP

5-question quizzes test your understanding. Pick an answer, get instant feedback with explanations, and earn XP for every correct answer.

  • Multiple choice with 4 options
  • Instant correct/wrong feedback
  • Explanation after each answer
  • XP reward for correct answers
09 / Review

Mistake Garden
Learn from Errors

Wrong answers become learning opportunities. Review each saved mistake, understand the explanation, and clear it when you've mastered it.

  • Auto-saves wrong answers
  • Full question + explanation review
  • Clear mistakes once mastered
  • Track improvement over time

User Profile
Track Your Growth

See your total XP, streak, achievements, and learning stats at a glance. Your profile tells the story of your preparation journey.

  • Score summary with breakdown

  • Data-driven study guidance

Search, view details, change roles, and deactivate user accounts. Full admin control over the user base.
