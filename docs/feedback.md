# Feedback & Iteration — NerdQuiz

This document tracks the feedback received during the Mid-point Review and the subsequent iterations made to the project.

## 📝 Feedback Cycle 1: Mid-point Review

**Reviewer**: Team Lead / Peer Reviewer
**Date**: July 2026

### 🚩 Critical Issues
- **Issue**: The "Learning Map" feels static.
- **Feedback**: "The map is just a list of boxes. It should feel more like a journey."
- **Action**: Plan to add Framer Motion animations and a 'path' visual to the map.

- **Issue**: Content volume is too low.
- **Feedback**: "Three topics are not enough to demonstrate the value of the app."
- **Action**: Seeded 8+ comprehensive topics covering Algorithms, OS, Networks, DBs, and Security.

### 💡 Improvement Suggestions
- **Suggestion**: "Add a way to see the correct answer immediately after a mistake."
- **Action**: Implemented the **Mistake Garden**, which specifically pairs the user's wrong answer with the correct one.

- **Suggestion**: "Make it feel more like a mobile app."
- **Action**: Added a mobile-first responsive design audit to the Week 2 priority list.
